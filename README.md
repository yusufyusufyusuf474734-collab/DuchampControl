# Duchamp Control

Root tabanlı sistem kontrol uygulaması — Poco X6 Pro 5G / Redmi K70E (MT6897 Dimensity 8300 Ultra)

## Özellikler

- CPU/GPU governor, frekans kilitleme
- Canlı izleme grafikleri (CPU, GPU, RAM, sıcaklık)
- Performans profilleri (Pil / Dengeli / Performans / Oyun)
- Batarya: şarj limiti, hızlı şarj kontrolü
- Termal sensörler
- Bellek: swappiness, ZRAM algoritması, I/O scheduler
- Ekran: HBM, DC Dimming, yenileme hızı
- Ses: Dolby Atmos profili, hoparlör/mikrofon gain
- Ağ: TCP congestion, DNS, Wi-Fi güç tasarrufu
- Dokunmatik polling rate (60/120/240/360Hz)
- IR Blaster test
- Kernel Tweaks (vm, net, fs parametreleri)
- Magisk / KernelSU modül yönetimi
- Uygulama yöneticisi (durdur, dondur, devre dışı, kaldır)
- Doze istisna listesi yönetimi
- Güvenlik: bootloader durumu, root log, hosts editörü
- Zamanlayıcı: saat/şarj/sıcaklık tetikleyicili profil otomasyonu
- Tema & kişiselleştirme: koyu/açık tema, 8 accent rengi
- Logcat görüntüleyici

## Gereksinimler

- Android 14+ (API 34)
- Magisk veya KernelSU (root)
- Poco X6 Pro 5G / Redmi K70E

## Kurulum

Android Studio ile açın ve cihaza yükleyin.
